bin_game_mvc
============

Run dependencies
----------------

 - Standart library of C language
 - Standart library of C++ language
 - lua
 - luabind

Build dependencies
------------------

 - All the run dependencies
 - cmake
 - make
 - C++ compiler

How to build
------------

The first command: `cmake /path/to/bin_game_mvc`

And the second (in directory where you need to build):
`make`
