bin_game_mvc
============

It's a console C++ version of [2048 game][2048].

Run dependencies
----------------

 - Standart library of C language
 - Standart library of C++ language
 - lua
 - luabind

Build dependencies
------------------

 - All the run dependencies
 - cmake
 - make
 - C++ compiler

How to build (Linux)
--------------------

```bash
$ mkdir build-dir
$ cd build-dir
$ cmake /path/to/bin_game_mvc
$ make

[2048]: https://gabrielecirulli.github.io/2048/
